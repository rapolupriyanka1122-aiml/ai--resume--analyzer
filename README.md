# AI Resume Analyzer

## 📖 Overview

AI Resume Analyzer is a Python-based web application that uses Artificial Intelligence to analyze resumes. It extracts text from PDF resumes, evaluates the content, identifies strengths and missing skills, and provides personalized suggestions to improve the resume for better job opportunities.

## ✨ Features

- Upload resumes in PDF format
- Extract text from resumes
- AI-powered resume analysis
- Identify key skills and missing skills
- Generate improvement suggestions
- User-friendly interface using Streamlit

## 🛠 Technologies Used

- Python
- Streamlit
- Google Gemini AI
- PyPDF2
- Natural Language Processing (NLP)

## 📂 Project Structure

```
AI_Resume_Analyzer/

```

## ▶️ How to Run

1. Clone the repository.
2. Install the required libraries:
   ```
   pip install -r requirements.txt
   ```
3. Add your Google Gemini API key.
4. Run the application:
   ```
   streamlit run app.py
   ```
5. Open the URL shown in your browser and upload a PDF resume.

## 🎯 Project Objective

The objective of this project is to help job seekers improve their resumes using Artificial Intelligence by providing detailed feedback, skill analysis, and recommendations.
